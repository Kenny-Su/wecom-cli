# 创建日程

最后更新：2025/11/19

该接口用于在日历中创建一个日程。

## 请求信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text
https://qyapi.weixin.qq.com/cgi-bin/oa/schedule/add?access_token=ACCESS_TOKEN
```

---

## 请求包体

```json
{
  "schedule": {
    "admins": [
      "admin1",
      "admin2"
    ],
    "start_time": 1571274600,
    "end_time": 1571320210,
    "is_whole_day": 1,
    "attendees": [
      {
        "userid": "userid2"
      }
    ],
    "summary": "需求评审会议",
    "description": "2.0版本需求初步评审",
    "reminders": {
      "is_remind": 1,
      "remind_before_event_secs": 3600,
      "remind_time_diffs": [0, -3600],

      "is_custom_repeat": 1,
```

## 参数说明

| 参数                                            | 必须 | 类型       | 说明                                      |
| --------------------------------------------- | -- | -------- | --------------------------------------- |
| `access_token`                                | 是  | string   | 调用接口凭证                                  |
| `schedule`                                    | 是  | obj      | 日程信息                                    |
| `schedule.admins`                             | 否  | string[] | 日程的管理员 userid 列表，管理员必须在共享成员列表中，最多指定 3 人 |
| `schedule.attendees`                          | 否  | obj[]    | 日程参与者列表，最多支持 1000 人                     |
| `schedule.attendees.userid`                   | 是  | string   | 日程参与者 ID，不多于 64 字节                      |
| `schedule.summary`                            | 否  | string   | 日程标题，0 ~ 128 字符。不填默认显示为“新建事件”           |
| `schedule.description`                        | 否  | string   | 日程描述，不多于 1000 个字符                       |
| `schedule.reminders`                          | 否  | obj      | 提醒相关信息                                  |
| `schedule.reminders.is_remind`                | 否  | int32    | 是否需要提醒。`0`-否；`1`-是                      |
| `schedule.reminders.is_repeat`                | 否  | int32    | 是否重复日程。`0`-否；`1`-是                      |
| `schedule.reminders.remind_before_event_secs` | 否  | uint32   | 日程开始前多少秒提醒，仅 `is_remind=1` 时有效          |
| `schedule.reminders.remind_time_diffs`        | 否  | int32[]  | 多个提醒时间配置，仅 `is_remind=1` 时有效            |
| `schedule.reminders.repeat_type`              | 否  | uint32   | 重复类型                                    |
| `schedule.reminders.repeat_until`             | 否  | uint32   | 重复结束时间（Unix 时间戳）                        |
| `schedule.reminders.is_custom_repeat`         | 否  | uint32   | 是否自定义重复。`0`-否；`1`-是                     |
| `schedule.reminders.repeat_interval`          | 否  | uint32   | 重复间隔                                    |
| `schedule.reminders.repeat_day_of_week`       | 否  | uint32[] | 每周周几重复                                  |
| `schedule.reminders.repeat_day_of_month`      | 否  | uint32[] | 每月哪几天重复                                 |
| `schedule.reminders.timezone`                 | 否  | uint32   | 时区（UTC 偏移量）                             |
| `schedule.location`                           | 否  | string   | 日程地址，不多于 128 个字符                        |
| `schedule.start_time`                         | 是  | uint32   | 日程开始时间，Unix 时间戳                         |
| `schedule.end_time`                           | 是  | uint32   | 日程结束时间，Unix 时间戳                         |
| `schedule.cal_id`                             | 否  | string   | 日程所属日历 ID                               |
| `schedule.is_whole_day`                       | 否  | uint32   | 是否为全天日程。`0`-否；`1`-是                     |
| `agentid`                                     | 否  | uint32   | 授权方安装的应用 agentid，仅旧第三方多应用套件需要填写         |

## reminders 字段详细说明

### `remind_before_event_secs`

表示在日程开始前多少秒提醒。

支持值：

| 值       | 说明        |
| ------- | --------- |
| `0`     | 事件开始时     |
| `300`   | 开始前 5 分钟  |
| `900`   | 开始前 15 分钟 |
| `3600`  | 开始前 1 小时  |
| `86400` | 开始前 1 天   |

---

### `remind_time_diffs`

提醒时间与 `start_time` 的差值，可配置多个提醒时间。

支持值：

| 值        | 说明        |
| -------- | --------- |
| `0`      | 事件开始时     |
| `-300`   | 开始前 5 分钟  |
| `-900`   | 开始前 15 分钟 |
| `-3600`  | 开始前 1 小时  |
| `-86400` | 开始前 1 天   |

当 `is_whole_day=1` 时，额外支持：

| 值         | 说明         |
| --------- | ---------- |
| `32400`   | 事件当天 09:00 |
| `-172800` | 开始前 2 天    |
| `-604800` | 开始前 1 周    |

> `remind_time_diffs` 与 `remind_before_event_secs` 同时存在时，以 `remind_time_diffs` 为准。

---

## 重复类型 `repeat_type`

| 值   | 说明  |
| --- | --- |
| `0` | 每日  |
| `1` | 每周  |
| `2` | 每月  |
| `5` | 每年  |
| `7` | 工作日 |

---

## 自定义重复说明

如果：

```text
is_custom_repeat = 0
```

系统会根据 `start_time` 和 `repeat_type` 自动推算重复时间。

例如：

* 本周三 8 点 + 每周重复 → 每周三 8 点重复
* 本月 3 号 10 点 + 每月重复 → 每月 3 号 10 点重复

如果：

```text
is_custom_repeat = 1
```

则可以：

* 使用 `repeat_day_of_week` 指定每周哪几天重复
* 使用 `repeat_day_of_month` 指定每月哪几号重复
* 使用 `repeat_interval` 指定重复间隔

例如：

| 配置                         | 含义         |
| -------------------------- | ---------- |
| `repeat_interval=3` + 每周重复 | 每 3 周重复一次  |
| `repeat_interval=3` + 每月重复 | 每 3 个月重复一次 |

---

## 时区 timezone

使用 UTC 偏移量表示：

* 东区为正数
* 西区为负数

例如：

```text
+8
```

表示北京时间东八区。

取值范围：

```text
-12 ~ +12
```

默认值为北京时间东八区。

# 取消日程

最后更新：2022/12/01

该接口用于取消指定的日程。

---

## 请求信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text
https://qyapi.weixin.qq.com/cgi-bin/oa/schedule/del?access_token=ACCESS_TOKEN
```

---

## 请求包体

```json
{
  "schedule_id": "17c7d2bd9f20d652840f72f59e796AAA",
  "op_mode": 1,
  "op_start_time": 1571274600
}
```

---

## 参数说明

| 参数              | 是否必须 | 类型     | 说明                         |
| --------------- | ---- | ------ | -------------------------- |
| `schedule_id`   | 是    | string | 日程 ID                      |
| `op_mode`       | 否    | uint32 | 操作模式（重复日程时有效）              |
| `op_start_time` | 否    | uint32 | 操作起始时间，仅 `op_mode=1/2` 时有效 |

---

# op_mode 操作模式

| 值   | 说明        |
| --- | --------- |
| `0` | 默认，删除所有日程 |
| `1` | 仅删除此日程    |
| `2` | 删除本次及后续日程 |

> 详见“重复日程的不同操作模式”。

---

# op_start_time 说明

仅当：

```text
op_mode = 1 或 2
```

时有效。

要求：

```text
op_start_time 必须是重复日程某一次周期的开始时间
```

格式：

```text
Unix 时间戳（精确到秒）
```

---

# 不同删除模式说明

## 1. 删除所有日程（默认）

即：

```text
op_mode = 0
```

效果：

```text
整个重复日程序列全部取消
```

---

## 2. 仅删除此日程

即：

```text
op_mode = 1
```

效果：

```text
只取消指定周期
其它周期保持不变
```

适用于：

* 某次会议取消
* 临时停课
* 单次活动取消

---

## 3. 删除本次及后续日程

即：

```text
op_mode = 2
```

效果：

```text
从指定周期开始
后续所有周期全部取消
```

适用于：

* 课程永久停开
* 周期性项目终止
* 长期会议结束

---

# 示例

## 删除某一次周期

假设：

```text
每周三 14:00 的重复日程
```

现在只取消：

```text
2022-09-14 14:00
```

请求：

```json
{
  "schedule_id": "17c7d2bd9f20d652840f72f59e796AAA",
  "op_mode": 1,
  "op_start_time": 1663135200
}
```

效果：

```text
仅取消 2022-09-14 这一期
其它周次仍然存在
```

---

## 删除未来所有周期

请求：

```json
{
  "schedule_id": "17c7d2bd9f20d652840f72f59e796AAA",
  "op_mode": 2,
  "op_start_time": 1663135200
}
```

效果：

```text
从 2022-09-14 开始
后续所有周期全部取消
```

---

# 返回结果

```json
{
  "errcode": 0,
  "errmsg": "ok"
}
```

---

# 返回参数说明

| 参数        | 说明    |
| --------- | ----- |
| `errcode` | 错误码   |
| `errmsg`  | 错误码说明 |

# 新增日程参与者

最后更新：2023/05/16

该接口用于在日历中为指定日程新增参与者列表。

> 注意：该接口是 **增量式更新**，不会覆盖已有参与者。

---

## 请求信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text id="x8f4kq"
https://qyapi.weixin.qq.com/cgi-bin/oa/schedule/add_attendees?access_token=ACCESS_TOKEN
```

---

## 请求包体

```json id="j5m1ra"
{
  "schedule_id": "17c7d2bd9f20d652840f72f59e796AAA",
  "attendees": [
    {
      "userid": "userid2"
    }
  ]
}
```

---

## 参数说明

| 参数                 | 必须 | 类型     | 说明                    |
| ------------------ | -- | ------ | --------------------- |
| `access_token`     | 是  | string | 调用接口凭证                |
| `schedule_id`      | 是  | string | 日程 ID，创建日程时返回         |
| `attendees`        | 否  | obj[]  | 日程参与者列表，累计最多支持 1000 人 |
| `attendees.userid` | 是  | string | 日程参与者 ID，不多于 64 字节    |

---

## 接口特点

该接口为：

```text id="u8r3pm"
增量式更新
```

即：

* 新参与者会被追加
* 原有参与者不会被覆盖
* 不需要传递完整参与人列表

适用于：

* 临时加人
* 会议补充参与者
* 动态扩展日程成员

---

## 返回结果

```json id="t9n4zb"
{
  "errcode": 0,
  "errmsg": "ok"
}
```

---

## 返回参数说明

| 参数        | 类型     | 说明    |
| --------- | ------ | ----- |
| `errcode` | int32  | 返回码   |
| `errmsg`  | string | 错误码描述 |

# 更新日程

最后更新：2023/10/24

该接口用于在日历中更新指定的日程。

---

## 注意事项

> 更新操作是 **覆盖式**，而不是增量式。

1. 如果需要增量更新成员，请使用：

   * “新增日程参与者”
   * “删除日程参与者”

2. 不可更新：

   * 创建者
   * 日程所属日历 ID

3. 如果修改字段与会议室有关联，则已预约会议室的日程无法通过此接口更新。

应用需先取消会议室预定，再更新日程。

与会议室关联的字段包括：

* `start_time`
* `end_time`
* `is_whole_day`
* `is_repeat`
* `repeat_type`
* `repeat_until`
* `is_custom_repeat`
* `repeat_interval`
* `repeat_day_of_week`
* `repeat_day_of_month`
* `timezone`

---

## 请求信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text id="n0q2ws"
https://qyapi.weixin.qq.com/cgi-bin/oa/schedule/update?access_token=ACCESS_TOKEN
```

## 参数说明

| 参数                                            | 必须 | 类型       | 说明                          |
| --------------------------------------------- | -- | -------- | --------------------------- |
| `access_token`                                | 是  | string   | 调用接口凭证                      |
| `skip_attendees`                              | 否  | uint32   | 是否不更新参与人。`0`-否；`1`-是。默认 `0` |
| `op_mode`                                     | 否  | uint32   | 操作模式（重复日程时有效）               |
| `op_start_time`                               | 否  | uint32   | 操作起始时间，仅 `op_mode=1/2` 时有效  |
| `schedule`                                    | 是  | obj      | 日程信息                        |
| `schedule.schedule_id`                        | 是  | string   | 日程 ID                       |
| `schedule.admins`                             | 否  | string[] | 日程管理员 userid 列表，最多 3 人      |
| `schedule.attendees`                          | 否  | obj[]    | 日程参与者列表，最多 1000 人           |
| `schedule.attendees.userid`                   | 是  | string   | 日程参与者 ID，不超过 64 字节          |
| `schedule.summary`                            | 否  | string   | 日程标题，0~128 字符               |
| `schedule.description`                        | 否  | string   | 日程描述，不超过 1000 字符            |
| `schedule.reminders`                          | 否  | obj      | 提醒相关信息                      |
| `schedule.reminders.is_remind`                | 否  | int32    | 是否提醒。`0`-否；`1`-是            |
| `schedule.reminders.is_repeat`                | 否  | int32    | 是否重复日程。`0`-否；`1`-是          |
| `schedule.reminders.remind_before_event_secs` | 否  | uint32   | 提前提醒秒数                      |
| `schedule.reminders.remind_time_diffs`        | 否  | int32[]  | 多提醒时间配置                     |
| `schedule.reminders.repeat_type`              | 否  | uint32   | 重复类型                        |
| `schedule.reminders.repeat_until`             | 否  | uint32   | 重复结束时间                      |
| `schedule.reminders.is_custom_repeat`         | 否  | uint32   | 是否自定义重复                     |
| `schedule.reminders.repeat_interval`          | 否  | uint32   | 重复间隔                        |
| `schedule.reminders.repeat_day_of_week`       | 否  | uint32[] | 每周周几重复                      |
| `schedule.reminders.repeat_day_of_month`      | 否  | uint32[] | 每月哪几天重复                     |
| `schedule.reminders.timezone`                 | 否  | uint32   | 时区                          |
| `schedule.location`                           | 否  | string   | 日程地址，不超过 128 字符             |
| `schedule.start_time`                         | 是  | uint32   | 日程开始时间（Unix 时间戳）            |
| `schedule.end_time`                           | 是  | uint32   | 日程结束时间（Unix 时间戳）            |
| `schedule.is_whole_day`                       | 否  | uint32   | 是否为全天日程                     |

---

## op_mode 操作模式

| 值   | 说明        |
| --- | --------- |
| `0` | 默认，修改全部周期 |
| `1` | 仅修改当前日程   |
| `2` | 修改将来的所有日程 |

---

## op_start_time 说明

仅当：

```text id="6k7x0m"
op_mode = 1 或 2
```

时有效。

该时间必须是重复日程中的某一次开始时间。

例如：

假设：

```text id="x33v8o"
start_time = 1661990950（2022-09-01 08:09:10）
```

并且：

```text id="j67ly6"
repeat_type = 每周重复
```

则 `op_start_time` 可为：

```text id="lhg9h1"
1661990950（2022-09-01 08:09:10）
1662595750（2022-09-08 08:09:10）
1663200550（2022-09-15 08:09:10）
```

等任意一个周期实例时间。

---

## reminders.remind_before_event_secs

支持值：

| 值       | 说明       |
| ------- | -------- |
| `0`     | 开始时      |
| `300`   | 提前 5 分钟  |
| `900`   | 提前 15 分钟 |
| `3600`  | 提前 1 小时  |
| `86400` | 提前 1 天   |

---

## reminders.remind_time_diffs

支持值：

| 值        | 说明       |
| -------- | -------- |
| `0`      | 开始时      |
| `-300`   | 提前 5 分钟  |
| `-900`   | 提前 15 分钟 |
| `-3600`  | 提前 1 小时  |
| `-86400` | 提前 1 天   |

当：

```text id="rbow0u"
is_whole_day = 1
```

时额外支持：

| 值         | 说明       |
| --------- | -------- |
| `32400`   | 当天 09:00 |
| `-172800` | 提前 2 天   |
| `-604800` | 提前 1 周   |

> 当 `remind_time_diffs` 有值时，会优先于 `remind_before_event_secs` 生效。

---

## repeat_type 重复类型

| 值   | 说明  |
| --- | --- |
| `0` | 每日  |
| `1` | 每周  |
| `2` | 每月  |
| `5` | 每年  |
| `7` | 工作日 |

---

## 自定义重复说明

如果：

```text id="s8xjlwm"
is_custom_repeat = 0
```

系统将根据：

* `start_time`
* `repeat_type`

自动推算下一次重复时间。

例如：

* 周三 08:00 + 每周重复 → 每周三 08:00
* 每月 3 号 10:00 + 每月重复 → 每月 3 号 10:00

如果：

```text id="t1vvbh"
is_custom_repeat = 1
```

则可结合：

* `repeat_day_of_week`
* `repeat_day_of_month`
* `repeat_interval`

进行自定义周期配置。

例如：

| 配置                         | 含义         |
| -------------------------- | ---------- |
| `repeat_interval=2` + 每周重复 | 每 2 周重复一次  |
| `repeat_interval=2` + 每月重复 | 每 2 个月重复一次 |

---

## timezone 时区说明

使用 UTC 偏移量表示：

* 东区为正
* 西区为负

例如：

```text id="5ow99p"
+8
```

表示北京时间（UTC+8）。

取值范围：

```text id="vzt90r"
-12 ~ +12
```

默认值为东八区。

---

## 特别说明

当：

```text id="r99hs2"
op_mode = 1 或 2
```

时：

* `start_time`
* `end_time`

必须是：

```text id="z90nxy"
op_start_time 当天或之后的时间
```

否则更新会失败。

---

## 返回结果

```json id="eh8w8z"
{
  "errcode": 0,
  "errmsg": "ok",
  "schedule_id": "17c7d2bd9f20d652840f72f59e796AAA"
}
```

---

## 返回参数说明

| 参数            | 类型     | 说明          |
| ------------- | ------ | ----------- |
| `errcode`     | int32  | 返回码         |
| `errmsg`      | string | 错误码描述       |
| `schedule_id` | string | 新产生的重复日程 ID |

> 对于重复日程，如果不是修改全部周期，系统会修剪原重复日程并生成新的重复日程，此时会返回新的 `schedule_id`。

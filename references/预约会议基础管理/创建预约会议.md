# 创建预约会议

**最后更新：2025/12/15**

该接口用于创建一个预约会议。

---

## 接口信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text
https://qyapi.weixin.qq.com/cgi-bin/meeting/create?access_token=ACCESS_TOKEN
```

---

## 请求包体

```json
{
  "admin_userid": "zhangsan",
  "title": "新建会议",
  "meeting_start": 1600000000,
  "meeting_duration": 3600,
  "description": "新建会议描述",
  "location": "广州媒体港",
  "agentid": 1000014,
  "invitees": {
    "userid": ["lisi", "wangwu"]
  },
  "settings": {
    "remind_scope": 1,
    "password": "1234",
    "enable_waiting_room": false,
    "allow_enter_before_host": true,
    "enable_enter_mute": 1,
    "enable_screen_watermark": false,
    "hosts": {
      "userid": ["lisi", "wangwu"]
    },
    "ring_users": {
      "userid": ["zhangsan", "lis"]
    }
  },
  "cal_id": "wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA",
  "reminders": {
    "is_repeat": 1,
    "repeat_type": 0,
    "repeat_until": 1606976813,
    "repeat_interval": 1,
    "remind_before": [0, 900]
  }
}
```

---

## 参数说明

| 参数                 | 必须 | 说明                                                                                                        |
| ------------------ | -- | --------------------------------------------------------------------------------------------------------- |
| `access_token`     | 是  | 调用接口凭证。获取方法查看“获取access_token”                                                                             |
| `admin_userid`     | 是  | 会议管理员 userid                                                                                              |
| `title`            | 是  | 会议标题，最多支持 40 个字节或 20 个 UTF-8 字符                                                                           |
| `meeting_start`    | 是  | 会议开始时间 Unix 时间戳，需大于当前时间                                                                                   |
| `meeting_duration` | 是  | 会议持续时间（秒），最小 300 秒，最大 86399 秒                                                                             |
| `description`      | 否  | 会议描述，最多支持 500 个字节或 UTF-8 字符                                                                               |
| `location`         | 否  | 会议地点，最多 128 个字符                                                                                           |
| `agentid`          | 否  | 授权方安装的应用 agentid，仅旧的第三方多应用套件需要填写                                                                          |
| `invitees`         | 否  | 邀请参会成员。任何 userid 不合法或不在应用可见范围内会直接报错。普通企业最多 100 人；付费企业根据在线会议室或高级账号容量决定，但最多 300 人。超过 300 人请调用“更新会议受邀成员列表”接口 |
| `invitees.userid`  | 否  | 参与会议的企业成员 userid                                                                                          |
| `cal_id`           | 否  | 会议所属日历 ID。该日历必须由 access_token 对应应用创建。第三方应用必须指定该参数。不超过 64 字节                                               |
| `settings`         | 否  | 会议配置，详见 [Settings](#settings-说明)                                                                          |
| `reminders`        | 否  | 重复会议相关配置，详见 [Reminders](#reminders-说明)                                                                    |

### cal_id 说明

* 若参与人在日历分享范围内，则插入到该日历（同时插入参与人的默认日历）
* 若不在分享范围内，则仅插入参与者默认日历
* 不填写时，默认插入参与者默认日历
* 第三方应用必须指定 `cal_id`

---

# Settings 说明

| 参数                        | 必须 | 说明                                                                           |
| ------------------------- | -- | ---------------------------------------------------------------------------- |
| `password`                | 否  | 入会密码，仅支持 4～6 位纯数字                                                            |
| `enable_waiting_room`     | 否  | 是否开启等候室：`true` 开启；`false` 不开启。默认关闭                                           |
| `allow_enter_before_host` | 否  | 是否允许主持人未入会前加入：`true` 允许；`false` 不允许。默认允许                                     |
| `remind_scope`            | 否  | 会议开始时来电提醒方式：`1` 不提醒；`2` 仅提醒主持人；`3` 提醒所有成员；`4` 指定部分成员响铃。默认仅提醒主持人              |
| `enable_enter_mute`       | 否  | 入会静音设置：`1` 开启；`0` 关闭；`2` 超过 6 人自动开启。默认超过 6 人自动开启                             |
| `enable_screen_watermark` | 否  | 是否开启屏幕水印：`true` 开启；`false` 不开启。默认关闭                                          |
| `hosts`                   | 否  | 主持人列表，最多 10 人。若包含 `creator_userid` 会自动过滤。任何 userid 不合法会直接报错。仅购买高级功能的企业可指定主持人 |
| `hosts.userid`            | 否  | 企业成员 userid                                                                  |
| `ring_users`              | 否  | 指定响铃成员列表。若 `remind_scope=4` 且为空，则所有成员均不响铃                                    |
| `ring_users.userid`       | 否  | 指定响铃成员 userid                                                                |

---

# Reminders 说明

| 参数                | 必须 | 说明                                           |
| ----------------- | -- | -------------------------------------------- |
| `is_repeat`       | 否  | 是否周期性会议：`1` 周期性会议；`0` 非周期性会议。默认 `0`          |
| `repeat_type`     | 否  | 周期重复类型：`0` 每天；`1` 每周；`2` 每月；`7` 每个工作日。默认 `0` |
| `repeat_until`    | 否  | 重复结束时间。周期性会议有效。若未设置或超出最大结束时间，则自动设置为最大结束时间    |
| `repeat_interval` | 否  | 重复间隔。目前仅 `repeat_type=1`（每周）时支持，且值不能大于 `2`   |
| `remind_before`   | 否  | 会议提醒时间（相对于 `meeting_start` 前的秒数）             |

### remind_before 支持值

| 值       | 说明       |
| ------- | -------- |
| `0`     | 会议开始时提醒  |
| `300`   | 提前 5 分钟  |
| `900`   | 提前 15 分钟 |
| `3600`  | 提前 1 小时  |
| `86400` | 提前 1 天   |

> 若指定了不支持的值，则表现为会议开始时提醒。

### 周期会议限制

* 每天 / 每工作日 / 每周：最多重复 200 次
* 每两周 / 每月：最多重复 50 次

---

# 权限说明

* 发起人和企业内部参与人必须在应用可见范围内
* 自建应用需要配置在“可调用接口的应用”列表中

---

# 返回结果

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "meetingid": "XXXXXXXXX",
  "excess_users": ["zhangsan", "lisi"]
}
```

---

## 返回参数说明

| 参数             | 说明                                      |
| -------------- | --------------------------------------- |
| `errcode`      | 返回码                                     |
| `errmsg`       | 返回码描述                                   |
| `meetingid`    | 会议 ID，可用于调用“进入会议”接口（包括小程序接口和 JS-SDK 接口） |
| `excess_users` | 包含无效会议账号的 userid，仅在专业版企业部分参会人无有效会议账号时返回 |

---

> 注：会议改为应用身份创建后，也允许指定之前应用创建的组织者身份日历。

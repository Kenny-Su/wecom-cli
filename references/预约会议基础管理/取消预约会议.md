# 取消预约会议

**最后更新：2023/04/26**

该接口用于取消一个指定的预约会议。

---

## 接口信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text id="m31kzp"
https://qyapi.weixin.qq.com/cgi-bin/meeting/cancel?access_token=ACCESS_TOKEN
```

---

## 请求包体

```json id="v4f2wy"
{
  "meetingid": "XXXXXXXXX"
}
```

---

## 参数说明

| 参数             | 必须 | 说明                             |
| -------------- | -- | ------------------------------ |
| `access_token` | 是  | 调用接口凭证。获取方法查看“获取 access_token” |
| `meetingid`    | 是  | 会议 ID，仅允许取消预约状态下的会议            |

---

# 权限说明

* 仅允许取消当前应用创建的会议
* 自建应用需要配置在“可调用接口的应用”列表中

---

# 返回结果

```json id="x9h7mq"
{
  "errcode": 0,
  "errmsg": "ok"
}
```

---

## 返回参数说明

| 参数        | 说明      |
| --------- | ------- |
| `errcode` | 返回码     |
| `errmsg`  | 返回码描述内容 |

# 获取会议详情

**最后更新：2025/07/28**

该接口用于获取指定会议的详情内容。

---

## 接口信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text id="3jxk9p"
https://qyapi.weixin.qq.com/cgi-bin/meeting/get_info?access_token=ACCESS_TOKEN
```

---

## 请求包体

```json id="x4q2rm"
{
  "meetingid": "XXXXXXXXX"
}
```

---

## 参数说明

| 参数             | 必须 | 说明                             |
| -------------- | -- | ------------------------------ |
| `access_token` | 是  | 调用接口凭证。获取方法查看“获取 access_token” |
| `meetingid`    | 是  | 会议 ID                          |

---

# 权限说明

* 只能拉取当前应用创建的会议
* 自建应用需要配置在“可调用接口的应用”列表中

---

# 返回参数说明

| 参数                 | 说明                                           |
| ------------------ | -------------------------------------------- |
| `errcode`          | 返回码                                          |
| `errmsg`           | 返回码描述                                        |
| `admin_userid`     | 会议管理员的 userid                                |
| `title`            | 会议标题，最大 60 字节                                |
| `meeting_start`    | 会议开始时间 Unix 时间戳                              |
| `meeting_duration` | 会议时长                                         |
| `description`      | 会议描述，最大 600 字节                               |
| `location`         | 会议地点，最多 128 个字符                              |
| `main_department`  | 发起人所在部门                                      |
| `status`           | 会议状态：`1` 待开始；`2` 会议中；`3` 已结束；`4` 已取消；`5` 已过期 |
| `attendees`        | 会议成员信息。快速会议仅返回已参与成员列表                        |
| `settings`         | 会议配置，详见 [Settings](#settings-说明)             |
| `reminders`        | 重复会议配置，详见 [Reminders](#reminders-说明)         |
| `meeting_code`     | 会议号                                          |
| `meeting_link`     | 入会链接                                         |

---

# attendees 说明

## attendees.member

企业内部成员列表。

| 参数                 | 说明                   |
| ------------------ | -------------------- |
| `userid`           | 企业内部成员 userid        |
| `status`           | 与会状态：`1` 已参与；`2` 未参与 |
| `first_join_time`  | 首次加入会议时间 Unix 时间戳    |
| `last_quit_time`   | 最后一次离开会议时间 Unix 时间戳  |
| `total_join_count` | 入会次数                 |
| `cumulative_time`  | 累计参会时长（秒）            |

---

## attendees.tmp_external_user

会中参会的外部联系人列表。

> 注意：微信入会用户不会返回。

| 参数                    | 说明                        |
| --------------------- | ------------------------- |
| `tmp_external_userid` | 外部用户临时 ID。同一用户在不同会议中的值不一致 |
| `status`              | 与会状态：`1` 已参与；`2` 未参与      |
| `first_join_time`     | 首次进入会议时间 Unix 时间戳         |
| `last_quit_time`      | 最后一次离开会议时间 Unix 时间戳       |
| `total_join_count`    | 入会次数                      |
| `cumulative_time`     | 累计参会时长（秒）                 |

### tmp_external_userid 说明

可通过“tmp_external_userid 转换接口”转换为 `external_userid`，用于识别外部参会人身份。

# Settings 说明

| 参数                        | 必须 | 说明                                                          |
| ------------------------- | -- | ----------------------------------------------------------- |
| `password`                | 否  | 入会密码，仅支持 4～6 位纯数字                                           |
| `enable_waiting_room`     | 否  | 是否开启等候室：`true` 开启；`false` 不开启。默认关闭                          |
| `allow_enter_before_host` | 否  | 是否允许成员在主持人入会前加入：`true` 允许；`false` 不允许。默认允许                  |
| `remind_scope`            | 否  | 来电提醒方式：`1` 不提醒；`2` 仅提醒主持人；`3` 提醒所有成员；`4` 指定部分成员响铃。默认提醒所有成员  |
| `enable_enter_mute`       | 否  | 入会静音设置：`1` 开启；`0` 关闭；`2` 超过 6 人自动开启。默认超过 6 人自动开启            |
| `enable_screen_watermark` | 否  | 是否开启屏幕水印：`true` 开启；`false` 不开启。默认关闭                         |
| `hosts`                   | 否  | 主持人列表，最多 10 人。若包含 `creator_userid` 会自动过滤。任何 userid 不合法会直接报错 |
| `hosts.userid`            | 否  | 企业成员 userid                                                 |
| `ring_users`              | 否  | 指定响铃成员列表                                                    |
| `ring_users.userid`       | 否  | 指定响铃成员 userid                                               |

---

# Reminders 说明

| 参数                | 必须 | 说明                                         |
| ----------------- | -- | ------------------------------------------ |
| `is_repeat`       | 否  | 是否周期性会议：`1` 周期性会议；`0` 非周期性会议。默认 `0`        |
| `repeat_type`     | 否  | 周期重复类型：`0` 每天；`1` 每周；`2` 每月；`7` 每个工作日      |
| `repeat_until`    | 否  | 重复结束时间。若未设置或超出最大结束时间，则自动设置为最大结束时间          |
| `repeat_interval` | 否  | 重复间隔。目前仅 `repeat_type=2`（每周）时支持，且值不能大于 `2` |
| `remind_before`   | 否  | 会议提醒时间（相对于 `meeting_start` 前的秒数）           |

### remind_before 支持值

| 值       | 说明       |
| ------- | -------- |
| `0`     | 会议开始时提醒  |
| `300`   | 提前 5 分钟  |
| `900`   | 提前 15 分钟 |
| `3600`  | 提前 1 小时  |
| `86400` | 提前 1 天   |

> 若指定了不支持的值，则表现为会议开始时提醒。

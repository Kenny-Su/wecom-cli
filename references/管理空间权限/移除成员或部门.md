# 移除成员/部门

> 最后更新：2024/07/19

该接口用于对指定空间移除成员/部门，操作者为应用本身。

## 请求方式

* **Method:** `POST`
* **Protocol:** `HTTPS`

## 请求地址

```text id="u4n8qx"
https://qyapi.weixin.qq.com/cgi-bin/wedrive/space_acl_del?access_token=ACCESS_TOKEN
```

---

# 请求包体

```json id="p6d1zr"
{
  "spaceid": "SPACEID",
  "auth_info": [
    {
      "type": 1,
      "userid": "USERID1"
    },
    {
      "type": 2,
      "departmentid": 1
    }
  ]
}
```

---

# 参数说明

| 参数             | 类型     | 是否必须 | 说明                                                   |
| -------------- | ------ | ---- | ---------------------------------------------------- |
| `spaceid`      | string | 是    | 空间 `spaceid`                                         |
| `auth_info`    | obj[]  | 是    | 被移除的空间成员信息                                           |
| `type`         | uint32 | 是    | 成员类型：`1` 个人，`2` 部门                                   |
| `userid`       | string | 是    | 成员 `userid`（`type=1` 时填写）                            |
| `departmentid` | uint32 | 是    | 部门 `departmentid`，32 位整型范围 `[0, 2^32)`（`type=2` 时填写） |

---

# 权限说明

* 自建应用需配置到“可调用应用”列表中的应用 `secret` 所获取的 `access_token` 来调用
* 第三方应用需具有“微盘”权限
* 代开发自建应用需具有“微盘”权限

---

# 返回示例

```json id="k2w7mv"
{
  "errcode": 0,
  "errmsg": "ok"
}
```

---

# 返回参数说明

| 参数        | 类型     | 说明    |
| --------- | ------ | ----- |
| `errcode` | int32  | 错误码   |
| `errmsg`  | string | 错误码说明 |

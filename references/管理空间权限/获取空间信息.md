# 获取空间信息

> 最后更新：2022/12/01

该接口用于获取空间信息，包括：空间成员、权限及安全设置。

## 请求方式

* **Method:** `POST`
* **Protocol:** `HTTPS`

## 请求地址

```text id="h3p8vx"
https://qyapi.weixin.qq.com/cgi-bin/wedrive/new_space_info?access_token=ACCESS_TOKEN
```

---

# 请求包体

```json id="k7m1zr"
{
  "spaceid": "SPACEID"
}
```

---

# 参数说明

| 参数        | 类型     | 是否必须 | 说明           |
| --------- | ------ | ---- | ------------ |
| `spaceid` | string | 是    | 空间 `spaceid` |

---

# 权限说明

* 自建应用需配置到“可调用应用”列表中的应用 `secret` 所获取的 `access_token` 来调用
* 第三方应用需具有“微盘”权限
* 代开发自建应用需具有“微盘”权限

# 返回参数说明

| 参数                                       | 类型       | 说明                      |
| ---------------------------------------- | -------- | ----------------------- |
| `errcode`                                | int32    | 错误码                     |
| `errmsg`                                 | string   | 错误码说明                   |
| `space_info`                             | obj      | 空间信息                    |
| `spaceid`                                | string   | 空间 ID                   |
| `space_name`                             | string   | 空间名                     |
| `auth_list`                              | obj      | 空间成员列表                  |
| `auth_info`                              | obj[]    | 空间成员信息                  |
| `type`                                   | uint32   | 成员类型：`1` 个人，`2` 部门      |
| `userid`                                 | string   | 成员 `userid`             |
| `departmentid`                           | uint32   | 部门 `departmentid`       |
| `auth`                                   | uint32   | 成员权限                    |
| `quit_userid`                            | string[] | 已退出空间或被移除权限的成员 `userid` |
| `space_sub_type`                         | uint32   | 空间类型，`0` 表示普通空间         |
| `secure_setting`                         | obj      | 空间安全设置                  |
| `enable_watermark`                       | bool     | 是否开启水印                  |
| `add_member_only_admin`                  | bool     | 是否仅管理员可添加成员             |
| `enable_share_url`                       | bool     | 是否启用邀请链接                |
| `share_url_no_approve`                   | bool     | 是否允许免审批加入               |
| `share_url_no_approve_default_auth`      | uint32   | 邀请链接默认权限                |
| `enable_share_external`                  | bool     | 是否允许对外分享                |
| `enable_share_external_admin`            | bool     | 是否仅管理员可对外分享             |
| `enable_space_add_external_member`       | bool     | 是否允许添加外部联系人成员           |
| `enable_space_add_external_member_admin` | bool     | 是否仅管理员可添加外部联系人成员        |
| `enable_confidential_mode`               | bool     | 是否开启保密模式                |
| `default_file_scope`                     | uint32   | 默认文件可见范围                |
| `create_file_only_admin`                 | bool     | 是否仅管理员可创建文件             |

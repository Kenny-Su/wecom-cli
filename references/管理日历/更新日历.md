# 更新日历

最后更新：2023/08/31

该接口用于修改指定日历的信息。

> 注意：更新操作是覆盖式，而不是增量式。

## 请求方式

* **Method**: `POST`
* **HTTPS URL**:

```text
https://qyapi.weixin.qq.com/cgi-bin/oa/calendar/update?access_token=ACCESS_TOKEN
```

---

## 请求包体

```json
{
  "skip_public_range": 0,
  "calendar": {
    "cal_id": "wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA",
    "admins": [
      "admin1",
      "admin2"
    ],
    "summary": "test_summary",
    "color": "#FF3030",
    "description": "test_describe_1",
    "shares": [
      {
        "userid": "userid1",
        "permission": 1
      },
      {
        "userid": "userid2",
        "permission": 3
      }
    ],
    "public_range": {
      "userids": ["abel", "jack"],
      "partyids": [1232, 34353]
    }
  }
}
```

---

## 请求参数说明

| 参数                               | 是否必须 | 说明                                        |
| -------------------------------- | ---- | ----------------------------------------- |
| `skip_public_range`              | 否    | 是否不更新可订阅范围。`0`：否；`1`：是。默认值为 `0`，即会更新可订阅范围 |
| `calendar`                       | 是    | 日历信息                                      |
| `calendar.cal_id`                | 是    | 日历 ID                                     |
| `calendar.admins`                | 否    | 日历管理员 `userid` 列表，最多指定 3 人                |
| `calendar.summary`               | 是    | 日历标题，长度 `1 ~ 128` 字符                      |
| `calendar.color`                 | 是    | 日历颜色，RGB 十六进制编码，例如 `#0000FF` 表示纯蓝色        |
| `calendar.description`           | 否    | 日历描述，长度 `0 ~ 512` 字符                      |
| `calendar.public_range`          | 否    | 公开范围，仅公共日历有效                              |
| `calendar.public_range.userids`  | 否    | 公开成员列表范围，最多指定 1000 个成员                    |
| `calendar.public_range.partyids` | 否    | 公开部门列表范围，最多指定 100 个部门                     |
| `calendar.shares`                | 否    | 日历通知范围成员列表，最多 2000 人                      |
| `calendar.shares.userid`         | 是    | 日历通知范围成员的 ID                              |
| `calendar.shares.permission`     | 否    | 日历通知范围成员权限，默认值为「可查看」                      |

### `shares.permission` 权限说明

| 值   | 权限      |
| --- | ------- |
| `1` | 可查看     |
| `3` | 仅查看闲忙状态 |

---

## 返回结果

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "fail_result": {
    "shares": [
      {
        "errcode": 40001,
        "errmsg": "not found",
        "userid": "userid3"
      }
    ]
  }
}
```

---

## 返回参数说明

| 参数                   | 说明            |
| -------------------- | ------------- |
| `errcode`            | 错误码           |
| `errmsg`             | 错误码说明         |
| `fail_result`        | 无效的输入内容       |
| `fail_result.shares` | 无效的日历通知范围成员列表 |
| `shares.errcode`     | 错误码           |
| `shares.errmsg`      | 错误码说明         |
| `shares.userid`      | 日历通知范围成员的 ID  |

# 删除日历

最后更新：2022/12/01

该接口用于删除指定日历。

## 请求方式

* **Method:** `POST`
* **Protocol:** `HTTPS`

## 请求地址

```text id="url-delete-calendar"
https://qyapi.weixin.qq.com/cgi-bin/oa/calendar/del?access_token=ACCESS_TOKEN
```

---

## 请求包体

```json id="request-delete-calendar"
{
  "cal_id": "wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA"
}
```

## 参数说明

| 参数       | 是否必须 | 说明   |
| -------- | ---- | ---- |
| `cal_id` | 是    | 日历ID |

---

## 返回结果

```json id="response-delete-calendar"
{
  "errcode": 0,
  "errmsg": "ok"
}
```

## 参数说明

| 参数        | 说明    |
| --------- | ----- |
| `errcode` | 错误码   |
| `errmsg`  | 错误码说明 |

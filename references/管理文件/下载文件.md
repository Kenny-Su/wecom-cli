# 下载文件

**最后更新：2022/12/01**

该接口用于下载文件。

## 请求信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text
https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_download?access_token=ACCESS_TOKEN
```

---

# 请求包体

```json
{
  "fileid": "FILEID",
  "selected_ticket": "SELECTED_TICKET"
}
```

---

# 参数说明

| 参数                | 类型     | 是否必须 | 说明                                                        |
| ----------------- | ------ | ---- | --------------------------------------------------------- |
| `fileid`          | string | 否    | 文件 `fileid`（只支持下载普通文件，不支持下载文件夹或微文档）                       |
| `selected_ticket` | string | 否    | 微盘和文件选择器 JSAPI 返回的 `selectedTicket`。若填写该参数，则无需填写 `fileid` |

---

# 注意事项

`fileid` 和 `selected_ticket` 必须填写且仅填写其中一组参数。

---

# 权限说明

* 自建应用需配置到“可调用应用”列表中的应用 `secret` 所获取的 `access_token` 来调用
* 第三方应用需具有“微盘”权限
* 代开发自建应用需具有“微盘”权限

---

# 返回示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "download_url": "DOWNLOAD_URL",
  "cookie_name": "COOKIE_NAME",
  "cookie_value": "COOKIE_VALUE"
}
```

---

# 返回参数说明

| 参数             | 类型     | 说明                   |
| -------------- | ------ | -------------------- |
| `errcode`      | int32  | 错误码                  |
| `errmsg`       | string | 错误码说明                |
| `download_url` | string | 下载请求 URL（有效期 `2` 小时） |
| `cookie_name`  | string | 下载请求携带的 Cookie Key   |
| `cookie_value` | string | 下载请求携带的 Cookie Value |

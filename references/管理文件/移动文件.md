# 移动文件

**最后更新：2024/07/19**

该接口用于将文件移动到指定位置。

## 请求信息

* **请求方式**：`POST`（HTTPS）
* **请求地址**：

```text
https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_move?access_token=ACCESS_TOKEN
```

---

# 请求包体

```json
{
  "fatherid": "FATHERID",
  "replace": true,
  "fileid": ["FILEID1", "FILEID2"]
}
```

---

# 参数说明

| 参数         | 类型     | 是否必须 | 说明                                                                                 |
| ---------- | ------ | ---- | ---------------------------------------------------------------------------------- |
| `fatherid` | string | 是    | 当前目录的 `fileid`，根目录时为空间 `spaceid`                                                   |
| `replace`  | bool   | 否    | 当目标目录中存在同名文件时是否覆盖。`true`：覆盖重名文件；`false`：自动进行冲突重命名处理（如 `xxx(1).txt`、`xxx(1).doc` 等） |
| `fileid`   | string | 是    | 文件 `fileid`                                                                        |

---

# 权限说明

* 自建应用需配置到“可调用应用”列表中的应用 `secret` 所获取的 `access_token` 来调用
* 第三方应用需具有“微盘”权限
* 代开发自建应用需具有“微盘”权限

---

# 返回示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "file_list": {
    "item": [
      {
        "fileid": "FILEID",
        "file_name": "FILE_NAME",
        "spaceid": "SPACEID",
        "fatherid": "FATHERID",
        "file_size": 11,
        "ctime": 17777777777,
        "mtime": 2777777777777,
        "file_type": 3,
        "file_status": 2,
        "sha": "SHA",
        "md5": "MD5"
      }
    ]
  }
}
```

---

# 返回参数说明

| 参数            | 类型     | 说明                                                                |
| ------------- | ------ | ----------------------------------------------------------------- |
| `errcode`     | int32  | 错误码                                                               |
| `errmsg`      | string | 错误码说明                                                             |
| `file_list`   | obj[]  | 移动文件的信息列表                                                         |
| `fileid`      | string | 文件 `fileid`                                                       |
| `file_name`   | string | 文件名字                                                              |
| `spaceid`     | string | 文件所在的空间 `spaceid`                                                 |
| `fatherid`    | string | 文件所在的目录 `fileid`，在根目录时为 `fileid`                                  |
| `file_size`   | uint64 | 文件大小                                                              |
| `ctime`       | uint64 | 文件创建时间                                                            |
| `mtime`       | uint64 | 文件最后修改时间                                                          |
| `file_type`   | uint32 | 文件类型：`1` 文件夹；`2` 文件；`3` 文档（文档）；`4` 文档（表格）；`5` 文档（收集表）；`6` 文档（幻灯片） |
| `file_status` | uint32 | 文件状态：`1` 正常；`2` 删除                                                |
| `sha`         | string | 文件 `sha`                                                          |
| `md5`         | string | 文件 `md5`                                                          |
